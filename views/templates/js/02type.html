변수와 자료형

변수variable와 상수constant

변수는 데이터를 담는 그릇 - var, let 으로 선언

상수는 변하지 않는 값을 담은 그릇 - const 으로 선언

변수/상수 선언시 자료형을 기술할 필요 x

변수 선언시 var 이나 let(ES6에서 추가)을 사용

실무에서는 오류를 최소화하기 위해 let 사용을 추천!

자바스크립트에서는 각 문장의 끝은 ;으로 마무리해야 함

자료형

자바스크립트 자료형에는 Number, String, Boolean, Null, Undefined, BigInt, 객체형등이 있음

숫자형

Number형으로 정수, 실수등을 다룰수 있음

BigInt형은 기존 정수보다 더 많은 범위의 수를 다룰수 있음

한편, Number형에는 일반적인 숫자외에 추가적으로 NaN(not a number), Infinity/-Infinity등의 특수숫자값을 사용할 수 있음

따라서, 말도 안되는 연산에도 오류는 발생하지 않음!

문자형

단일문자나 단일문자가 여러개 모인 문자열을 다룰때 사용

문자열을 정의할때는 '', "", `` 등을 이용함

Boolean형

논리형의 일종으로 true, false 등의 논리값만 존재

주로 긍정(true), 부정(false)을 나타내는 값에 사용

Null

null 값만 다루는 자료형

null은 존재하지 않는 값, 비어 있는 값, 알 수 없는 값을 나타내는데 사용

undefined

undefined 값만 다루는 자료형

undefined는 변수는 선언했지만, 값이 할당되지 않은 상태를 나타내는데 사용
