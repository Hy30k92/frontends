배열

개요

동일한 자료형의 자료를 여러개 저장하기 위한 자료구조

배열 생성시 [] 또는 Array()를 이용함

데이터의 입력순서를 기억함 : 순차 자료구조

요소 : 배열 안에 저장된 개별 데이터

인덱스 : 요소가 위치한 순서를 나타내는 숫자

인덱스는 0부터 시작함

자료구조 : 데이터를 구조적으로 표현하는 방식과 이를 구현하는데 필요한 알고리즘을 논하는 이론

즉, 데이터를 효율적으로 다룰수 있게 도와주는 데이터 보관방법과 데이터에 관한 연산의 집합체임

forEach 콜백

배열에 저장된 각 요소에 대해 제공된 콜백함수를 한번씩 실행하는 배열 메서드

배열의 요소를 순회하면서 특정 작업을 수행할 때 유용

콜백 callback : 특정 이벤트나 시점에 도달했을 때 호출되는 함수.
비동기 프로그래밍의 중요한 요소 중에 하나이고, 자바스크립트의 핵심
