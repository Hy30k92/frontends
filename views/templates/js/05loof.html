반복문

개요

조건을 만족하는 동안 코드를 반복적으로 수행하는 문장

자바스크립트에서는 기본적으로 3가지 반복문(for, while, do-while)이 지원되고, 그외 for-loop, for-in과 ES6에서 추가된 for-of등이 있음

for

조건식이 참 일 동안 코드블록을 반복적으로 실행

for(초기식; 판별식; 증감식) {

반복구문

}


무한 반복문 - for

for문의 경우 초기식,판별식,증감식을 비워두면 반복을 끊임없이 실행하는 무한반복문이 됨

중첩 반복문 - for

2개 이상의 for문을 겹쳐 사용하는 반복문

복잡한 작업을 반복 실행할때 유용하게 사용

break

원하는 시점에 반복문이나 코드블록에서 벗어나기 위해 사용

for-in

열거 가능한 객체의 속성을 순회해서 값을 다루고 싶을 때 사용하는 반복문

for (변수 in 객체) {

반복 실행할 코드

}

for-of

ES6에 추가된 반복문으로, 배열,문자열,Map,Set 등의 이터러블 객체의 값을 순차적으로 순회할 때 사용하는 반복문

이터러블 객체 : 순차적으로 값들이 저장된 객체를 의미, 따라서, 데이터 처리시 순서가 중요시 됨

for (변수 of 이터러블객체) {

반복 실행할 코드

}
