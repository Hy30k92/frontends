CSS flex

개요

뷰포트나 요소의 크기가 불명확하거나 동적으로 변할때에도 효율적으로 요소를 배치,정렬,분산할 수 있는 방법을 제공하는 CSS3의 새로운 레이아웃 방식

flexbox layout의 장점은 '복잡한 계산없이 요소의 크기와 순서를 유연하게 배치할 수 있다라는 것임

정렬,방향,순서,크기등을 유연하게 조절할 수 있기 때문에 별도의 분기처리를 줄일수 있고, CSS만으로 다양한 레이아웃 구현가능

기본 개념

Flex Container: Flexbox 레이아웃을 적용할 부모 요소

Flex Items: Flex Container 내부의 자식 요소들

Flex Container 속성

display: flex; 또는 display: inline-flex;로 설정

flex-direction: 주 축의 방향 설정 (row, column, row-reverse, column-reverse)

flex-wrap: 아이템 줄바꿈 설정 (nowrap, wrap, wrap-reverse)

justify-content: 주 축 방향으로 아이템 정렬 (flex-start, flex-end, center, space-between, space-around)

align-items: 교차 축 방향으로 아이템 정렬 (stretch, flex-start, flex-end, center, baseline)

align-content: 여러 줄일 때 교차 축 방향 정렬 (stretch, flex-start, flex-end, center, space-between, space-around)

Flex Items 속성

flex-grow: 남은 공간을 차지하는 비율

flex-shrink: 공간이 부족할 때 줄어드는 비율

flex-basis: 아이템의 기본 크기

flex: grow, shrink, basis를 한 번에 지정 (예: flex: 1 1 auto;)

align-self: 개별 아이템의 교차 축 정렬 (auto, flex-start, flex-end, center, baseline, stretch)

order: 아이템의 순서 변경
