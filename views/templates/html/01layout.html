정의

웹 페이지의 구조와 내용을 정의하는 마크업 언어를 의미

웹 브라우저에 의해 해석되고 사용자에게 표시될 문서의 구조를 정의

하이퍼텍스트를 이용해서 웹 페이지간의 이동을 가능하게 하는 링크도 생성 가능

요소와 태그, 속성

태그 : HTML 문서에 사용하는 마크업 구문을 의미

꺽쇠 괄호에 둘러싸여 있음 - <>

시작태그와 종료태그로 구성

요소 : 시작태그,내용,종료태그 모두를 포함한 전체 구조를 의미

html 문서의 구조적 단위를 지칭

속성 : 해당 태그에 추가 정보를 제공하거나 동작을 한정하는데 사용

사용 예 : 태그에 이름을 부여, 링크에 참조위치 지정, 이미지의 보조설명 선언 등등

주석

태그나 요소에 대한 설명으로 작성하는 코드

브라우저에 의해 시각적요소로 나타나지는 않음
