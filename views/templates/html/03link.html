링크 관련 태그

개요

html 에서 다른 문서로의 연결/이동 을 위해 작성하는 요소

사용자가 링크를 클릭하면 다른 웹페이지나 이메일주소, 파일등으로 이동할 수 있게 해줌

링크를 하이퍼(hyper)링크라고도 부름

링크를 만들려면 a 태그 내에 href 라는 속성에 문서내 특정위치, 이동할 다른 페이지나 외부 사이트를 지정하고, a 태그 사이에 링크를 설명할 텍스트를 포함하면 됨

링크의 특성 = 링크의 기본색 : 파랑 / 클릭한 링크색 : 보라색 / 링크를 눌렀을때색 : 빨강 , 단, CSS 를 이용하면 특성 변경 가능

문서 내 특정 위치로 이동

링크를 책갈피로도 생성할 수 있는데, id라는 속성 으로 책갈피이름을 지정하고 href 속성에 '♯책갈피 이름' 을 지정하면, 문서 내 지정한 위치로 이동할 수 있음

또한, href 속성에 '문서명♯책갈피 이름' 을 지정하면, 특정 파일의 책갈피로도 이동할 수 있음
