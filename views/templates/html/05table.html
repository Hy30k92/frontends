테이블 관련 태그

개요

여러 종류의 데이터를 일목요연하게 2차원 형태의 표로 나타내기 위해 사용하는 요소

table 태그로 테이블 시작을(표삽입),
tr 태그로 테이블의 행을,
th태그로 테이블의 열을,
td태그로 테이블의 일반셸을,
thead 태그로 테이블의 머릿글을,
tbody 태그로 테이블의 본문을 나타냄

테이블의 각 컬럼에 공통적인 속성을 정의하려면 colgroup과 col태그를 사용해야 함

테이블의 디자인적인 요소들은 CSS로 다루는 것을 추천함!
